Public Instance Properties

ConnectMode Get/Set the connect mode
ControlPort Get/set the controlPort. Can only be set if not currently connected
DeleteOnFailure Get/set the delete on failure flag
FTPFileFactory Override the chosen file factory with a user created one - meaning that a specific parser has been selected
LastValidReply Gets the latest valid reply from the server
RemoteHost Get/set the name of the remote host.
StrictReturnCodes Strict checking of return codes. If it is on (the default), all return codes must exactly match the expected code. If strict checking is off, only the first digit must match
Timeout Set the TCP timeout on the underlying socket.
TransferBufferSize Get/set the size of the buffers used in writing to and reading from the data sockets
TransferNotifyInterval Get the bytes transferred between each notification of the BytesTransferred event. Reduce this value to receive more frequent notifications of transfer progress
TransferType Get or set the current transfer type

Public Instance Methods

CancelResume Cancel the resume. Use this method if something goes wrong and the server is left in an inconsistent state
CancelTransfer Cancels the current transfer. Generally called from a separate thread. Note that this may leave partially written files on the server or on local disk, and should not be used unless absolutely necessary. The server is not notified
ChDir Change the remote working directory to that supplied
Connect Connect to the remote host. Cannot be currently connected. RemoteHost property must be set
DebugResponses Switch Debug of responses on or off
Delete Delete the specified remote file
DirOverloaded. List a directory's contents as an array of strings. A detailed listing is available, otherwise just filenames are provided. The detailed listing varies in details depending on OS and FTP server. Note that a full listing can be used on a file name to obtain information about a file
DirDetails List a directory's contents as an array of FTPFile objects. Should work for Windows and most Unix FTP servers - let us know about unusual formats (support@enterprisedt.com)
Equals (inherited from Object)Determines whether the specified Object is equal to the current Object.
Features Get the server supplied features
GetOverloaded. Get data from the FTP server.
GetHashCode (inherited from Object)Serves as a hash function for a particular type, suitable for use in hashing algorithms and data structures like a hash table.
GetSystem Get the type of the OS at the server
GetType (inherited from Object)Gets the Type of the current instance.
Help Get the help text for the specified command
Login Login into an account on the FTP server. This call completes the entire login process
MkDir Create the specified remote working directory
ModTime Get modification time for a remote file
Password Supplies the password for a previously supplied username to log into the FTP server. Must be preceeded by the user() method
PutOverloaded. Put data onto the FTP server. It is placed in the current directory. Allows appending if current file exists
Pwd Get the current remote working directory
Quit Quit the FTP session
QuitImmediately Quit the FTP session immediately by closing the control socket without sending the QUIT command
Quote Issue arbitrary ftp commands to the FTP server.
Rename Rename a file or directory
Resume Make the next file transfer (put or get) resume. For puts(), the bytes already transferred are skipped over, while for gets(), if writing to a file, it is opened in append mode, and only the bytes required are transferred. Currently resume is only supported for BINARY transfers (which is generally what it is most useful for).
RmDir Delete the specified remote working directory
Site Run a site-specific command on the server. Support for commands is dependent on the server
Size Get the size of a remote file. This is not a standard FTP command, it is defined in "Extensions to FTP", a draft RFC (draft-ietf-ftpext-mlst-16.txt)
ToString (inherited from Object)Returns a String that represents the current Object.
User Supply the user name to log into an account on the FTP server. Must be followed by the password() method - but we allow for
ValidateTransfer Validate that the Put() or get() was successful. This method is not for general use.
