FTPClient.DeleteOnFailure Property

Get/set the delete on failure flag

public bool DeleteOnFailure {get; set;}

Remarks

If true, a partially downloaded file is deleted if there is a failure during the download. For example, the connection to the FTP server might have failed. If false, the partially downloaded file remains on the client machine - and the download may be resumed, if it is a binary transfer. By default this flag is set to true.
